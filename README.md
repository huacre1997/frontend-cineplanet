# Frontend Cineplanet

This project is a auth application for Cineplanet

## Technologies Used

- TypeScript
- React
- Vite
- JWT

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/huacre1997/frontend-cineplanet.git
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```
